# Employment and Unemployment Analysis 

## Project Objective
Analyze employment and unemployment trends across countries and identify the impact of major global events like the 2008 financial crisis and COVID-19.

## Datasets Used
- employed_intnshp.csv
- unemployed_intnshp.csv

## Tasks Performed
- COVID recovery time analysis
- Pre-2008 baseline analysis
- Pre-COVID baseline analysis
- Crisis peak identification
- Trend analysis for employment

## Key Insights
- Unemployment spiked in 2009 and 2020
- COVID caused the sharpest increase in unemployment
- Recovery took around 1–3 years
- Employment increased over the long term for most countries

## Tools Used
- Python
- Pandas
- Matplotlib
